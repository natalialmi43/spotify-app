# Spotify-app
The use is for learn the concepts of Clean Arquitecture and Singleton Patron

# Learning objectives
Clean Architecture: Separating the project into Domain, Data, and Presentation layers. 

SOLID principles; Implementing interfaces to decouple code.

Design Patterns: Implementing the Singleton Pattern to ensure a single point of access for shared resources

# Project structure
The proyect structure following a Clean Architecture

- Domain: Contains the business logic (Entities and UseCases).
  
- Data: Handles data fetching from the API and local sources.
  
- Presentation: Manages the UI components (Views).
